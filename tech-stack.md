## Tech Stack

### Frontend
- **React** + **TypeScript**
- Tailwind
- React Router

### Backend
- **Node.js** + **Express** + **TypeScript**
- Database sessions for authentication

### Database
- **PostgreSQL** with **pgvector** extension (for AI knowledge base embeddings)

### ORM
- **Prisma**

### Email
- **SendGrid** or **Mailgun** — for sending AI responses and agent notifications

### AI
- **Claude API** (Anthropic) — response generation, ticket classification, summaries, and suggested replies
- Model: `claude-sonnet-4-6`
